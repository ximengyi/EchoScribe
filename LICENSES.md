# Third-Party Notices

EchoScribe uses third-party components. Keep this file updated when dependencies change.

## ffmpeg

The packaged `ffmpeg.exe` is used for media extraction and conversion. The binary copied during development may come from ShareX. Review the exact ffmpeg build license before public distribution, because enabled codecs and build flags can affect redistribution requirements.

## faster-whisper

EchoScribe uses `faster-whisper` for transcription. See the upstream project for license details:

https://github.com/SYSTRAN/faster-whisper

## Whisper Model

The bundled model is `Systran/faster-whisper-small`. Review its model card and license before public distribution:

https://huggingface.co/Systran/faster-whisper-small
